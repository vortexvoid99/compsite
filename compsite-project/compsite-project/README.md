# Barrow Comps (Cloudflare Pages Ready)

This repo is set up as a Vite + React + TypeScript + Tailwind (v4) SPA, ready to deploy on Cloudflare Pages.

## Cloudflare Pages settings
- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`

## SPA routing
`public/_redirects` contains the SPA fallback.
